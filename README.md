# Cloud Projects

This repository contains a collection of Amazon Web Services (AWS) related projects focused on cloud monitoring, resource management, deployment, and DevOps practices.

## Purpose

These projects demonstrate practical AWS workflows using Python, Flask, Boto3, and cloud deployment concepts. The goal is to build hands-on experience with core AWS services and real-world infrastructure scenarios.

## Project Overview

### 1. AWS EC2 Monitoring Project
Location: [AWS_Monitor_EC2_instance](AWS_Monitor_EC2_instance)

A Flask-based application that uses Boto3 to monitor and manage Amazon EC2 instances. It supports creating, starting, stopping, terminating, and syncing EC2 resources.

### 2. AWS S3 Bucket Monitoring Project
Location: [AWS_Monitor_S3_Bucket](AWS_Monitor_S3_Bucket)

A Flask and Boto3 application for monitoring and managing Amazon S3 buckets. It includes features for bucket creation, syncing, upload handling, and deletion workflows.

### 3. AWS Travel Memory Deployment Assignment
Location: [AWS_Travel_Memory_Deployment_Assignment](AWS_Travel_Memory_Deployment_Assignment)

An AWS deployment assignment focused on hosting a web application on EC2, configuring networking, security, and application architecture for production-style deployment.

## Technologies Used

- Python
- Flask
- Boto3
- MySQL
- AWS EC2
- AWS S3
- Nginx
- Linux/Ubuntu environments

## Prerequisites

Before running the projects, make sure you have:

- An AWS account
- AWS CLI configured with valid credentials
- Python 3 installed
- MySQL (for projects that use databases)
- Basic knowledge of AWS services such as EC2 and S3

## Getting Started

1. Clone this repository.
2. Navigate to the project folder you want to explore.
3. Review the project-specific README file for setup instructions.
4. Configure your AWS credentials and environment variables before running the application.

## Notes

Each project folder contains its own README with detailed setup steps, features, and usage instructions. This repository is intended as a learning and demonstration hub for AWS-based cloud development.
